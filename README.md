# Simple Interest Calculator

This is a simple interest calculator that calculates interest on a principal amount over a specific period at a given interest rate.

## How to use
1. Clone the repository:
    ```bash
    git clone https://github.com/muhammedShadi/simple-interest-calculator.git
    ```
2. Run the program:
    ```bash
    python simple_interest_calculator.py
    ```
